        \item C tells A about its route, A tells E, \ldots, D tells C updated distance
        \item then C tells A update distance, repeat
        \end{itemize}
    \item keeps increasing distance with each run around loop
    \end{itemize}
\item A--B--C is another possible loop
\end{itemize}
